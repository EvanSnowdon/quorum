# Example engagements

Abridged sample deliverables showing the standard Quorum report format:
conclusion-first structure (Pyramid Principle), `[DATA]` / `[INFERENCE]`
labeling with confidence scores, scenario analysis, a mandatory dissenting
view from the red team, and a data & confidence appendix.

| File | Engagement | Depth |
|---|---|---|
| [`cn-electric-vehicles.md`](cn-electric-vehicles.md) | Electric vehicles · China | scan |
| [`us-cybersecurity.md`](us-cybersecurity.md) | Cybersecurity services · United States | scan |

> These examples are format references, abridged from full runs. Figures are
> illustrative of how sourced claims are rendered and labeled; do not reuse
> them as research. Run your own engagement for current numbers:
>
> ```bash
> quorum --region CN --industry "electric vehicles" --depth standard
> ```
